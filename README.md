# Donald's Bukka 🍽️
A Nigerian restaurant menu web app built with React and Vite.

## Live Demo
👉 [donalds-bukka-react.vercel.app](https://donalds-bukka-react.vercel.app/)

## Features
- Browse authentic Nigerian dishes by category
- Add items to cart
- Adjust quantities in cart
- View order total

## Technologies Used
- React
- Vite
- JavaScript
- Tailwind CSS
- Vercel (deployment)
